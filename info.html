접근 권한에 대한 내용

이 서비스에서 제공하는 기능을 사용하려면 사용자 휴대전화에서 제공하는 권한에 대한 접근이 허용되어야 합니다. 각 권한이 앱이 제공하는 기능과 어떻게 연관되어 있는지, 보다 자세한 용도에 대하여 아래에서 항목별로 설명하고 있습니다.

- 카메라 접근 : AR모드 사용을 위해 필요합니다.


서드파티 API 이용약관 인정

이 서비스를 사용하는 것은 사용자가 이 서비스에 사용된 모든 서드파티 API의 이용약관에 함께 동의하며, 귀하에게 적용될 수 있는 서드파티 API의 약관에 귀속되는 것을 인정합니다.

이 서비스는 다음과 같은 목적으로 서드파티 API를 사용하고 있습니다.

- Google Analytics For Web : 서비스 트래픽 추적 및 보고를 통해 안정적인 서비스를 제공하기 위해 노력하고 있습니다.
